## 🗺️ Как получить координаты достопримечательностей

### Способ 1: Через Yandex Maps
1. Откройте Yandex Maps в браузере
2. Найдите достопримечательность на карте
3. Кликните правой кнопкой мыши → "Что здесь?"
4. Скопируйте координаты (они будут в формате: 55.755826, 37.617299)

### Способ 2: Через Google Maps
1. Откройте Google Maps
2. Найдите достопримечательность
3. Кликните правой кнопкой → "Что здесь?"
4. Координаты появятся внизу экрана

### Способ 3: Если знаете адрес
Могу помочь найти координаты по адресу.

## 📁 Что делать с медиа файлами (фото и аудио)

### Вариант 1: Отправить файлы отдельно
- Создайте архив (ZIP) с фотографиями и аудио
- Назовите файлы по порядку: `attraction_1.jpg`, `attraction_1.mp3`, `attraction_2.jpg`, и т.д.

### Вариант 2: Добавить позже
- Сначала можно добавить только текстовые данные
- Медиа файлы добавим позже

## 🚀 Что я сделаю после получения данных

1. ✅ Создам JSON файл с данными маршрута
2. ✅ Добавлю код для загрузки данных в базу данных при первом запуске
3. ✅ Обновлю логику, чтобы маршруты загружались из локальной БД
4. ✅ Если есть медиа файлы - добавлю их в проект
5. ✅ Протестирую отображение маршрута на карте

---

## 💡 Важные замечания

- **Polyline маршрута** можно добавить позже или я помогу его создать по координатам остановок
- **Медиа файлы** (фото и аудио) можно добавить позже, сначала добавим текстовые данные
- **Координаты** должны быть точными (минимум 6 знаков после запятой)
- **Порядок** достопримечательностей должен соответствовать порядку следования по маршруту
- **Остановки** - это список названий остановок по порядку следования маршрута
- **История маршрута** и **Описание достопримечательностей** - это тексты, которые будут показаны на экране описания маршрута перед переходом к карте
- Если какое-то поле не заполнено, можно оставить пустым - оно просто не будет отображаться
