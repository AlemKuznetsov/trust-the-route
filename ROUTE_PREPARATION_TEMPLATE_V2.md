# Шаблон для подготовки маршрута (с поддержкой нескольких изображений)

Используйте этот шаблон для подготовки данных маршрута. После заполнения данных я создам JSON файл и добавлю его в проект.

---

## 📝 Инструкция по заполнению

1. **Скопируйте шаблон ниже**
2. **Заполните все поля** (можно оставить пустыми, если информации нет)
3. **Для каждой достопримечательности укажите 3-4 фотографии** (можно указать имена файлов или добавить позже)
4. **Отправьте заполненный шаблон** - я создам JSON файл и добавлю его в проект

## 📁 Структура медиа файлов

### Рекомендуемая структура именования:

Для достопримечательности с ID `attraction_1`:
- `attraction_1_photo_1.jpg` - первое фото
- `attraction_1_photo_2.jpg` - второе фото
- `attraction_1_photo_3.jpg` - третье фото
- `attraction_1_photo_4.jpg` - четвертое фото (если есть)
- `attraction_1.mp3` - аудиофайл

Или можно использовать более описательные имена:
- `attraction_1_main.jpg` - главное фото
- `attraction_1_detail_1.jpg` - деталь 1
- `attraction_1_detail_2.jpg` - деталь 2
- `attraction_1_interior.jpg` - интерьер

### Куда сохранять файлы:

**Фотографии:**
- Папка: `app/src/main/assets/images/`
- Формат: JPG или PNG
- Рекомендуемый размер: не более 2MB на фото

**Аудио:**
- Папка: `app/src/main/assets/audio/`
- Формат: MP3, OGG или M4A
- Рекомендуемая длительность: 1-3 минуты

## 💡 Важные замечания

- **ID маршрута** должен быть уникальным (латинские буквы, цифры, подчеркивания)
- **Координаты** должны быть точными (минимум 6 знаков после запятой)
- **Порядок** достопримечательностей должен соответствовать порядку следования по маршруту
- **Фотографии**: можно указать 1-4 фото на каждую достопримечательность
- **Имена файлов**: используйте понятные имена, связанные с ID достопримечательности
- **Polyline** можно добавить позже или я помогу создать по координатам остановок
- Медиа файлы можно добавить позже, сначала добавьте текстовые данные
