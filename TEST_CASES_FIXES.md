## Известные проблемы и ограничения

1. **Размер шрифта:** Изменения размера шрифта применяются через MaterialTheme, поэтому некоторые кастомные текстовые стили могут не учитывать выбранный размер.

2. **Производительность:** При изменении размера шрифта может происходить пересоздание всех экранов для применения новой типографии.

---

## Примечания для тестирования

1. Для проверки размера шрифта рекомендуется использовать разные экраны приложения (главное меню, маршруты, карта, настройки).

2. Для проверки экрана "Информация о приложении" убедитесь, что вы нажимаете именно на раздел "Информация о приложении" в настройках, а не на "О нас" (если такой раздел существует).

3. Для проверки проблемы с показом экрана авторизации рекомендуется:
   - Авторизоваться в приложении
   - Полностью закрыть приложение
   - Запустить приложение снова и внимательно наблюдать за первым экраном

4. При тестировании размера шрифта обратите внимание на:
   - Заголовки экранов
   - Текст в карточках
   - Текст на кнопках
   - Описания и подсказки
