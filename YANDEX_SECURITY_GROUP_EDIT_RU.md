# Редактирование Security Group в Yandex Cloud

## Ваша Security Group: `default-sg-enpgpegelia72k65sf2p`

## Как открыть порт 8080 (БЕЗ остановки VM):

### Шаг 1: Найдите Security Group

1. **В левом меню консоли** найдите раздел **"VPC"** (Virtual Private Cloud)
   - Или "Сеть" → "VPC"

2. **Перейдите в "Security Groups"** (Группы безопасности)

3. **Найдите группу**: `default-sg-enpgpegelia72k65sf2p`
   - Можно использовать поиск в списке

4. **Нажмите на название группы** для открытия

### Шаг 2: Добавьте правило для порта 8080

1. **В открывшейся странице Security Group** найдите раздел **"Правила"** или **"Rules"**

2. **Нажмите "Добавить правило"** или **"Add rule"**

3. **Настройте правило:**

   **Вкладка "Входящий трафик" (Inbound):**
   - **Направление**: Входящий (Inbound)
   - **Протокол**: TCP
   - **Порты**: `8080`
   - **Источник**: CIDR
   - **CIDR блоки**: `0.0.0.0/0`
     - (Или укажите ваш IP для большей безопасности)
   - **Описание**: "Backend API порт 8080"

4. **Нажмите "Сохранить"** или **"Save"**

### Шаг 3: Проверка

Правило применится автоматически без перезапуска VM!

Проверьте доступность порта:

```bash
# С вашего компьютера (PowerShell)
Test-NetConnection -ComputerName 158.160.217.181 -Port 8080
```

## Альтернативный способ (если не нашли раздел VPC):

1. **В консоли Yandex Cloud** вверху есть поиск
2. **Введите**: "Security Groups" или "Группы безопасности"
3. **Откройте найденный раздел**
4. **Найдите** `default-sg-enpgpegelia72k65sf2p`
