# Assets папка

Здесь будут храниться:
- JSON файлы с данными маршрутов
- Изображения достопримечательностей (опционально)
- Аудиофайлы (опционально)

## Структура:

```
assets/
├── routes/
│   ├── bus_b.json          (данные маршрута "Автобус Б")
│   └── ...
├── images/                 (опционально, если не используем res/drawable)
│   └── ...
└── audio/                 (опционально, если не используем res/raw)
    └── ...
```
