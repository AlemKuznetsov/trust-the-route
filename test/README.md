# Тесты для приложения Trust The Route

Эта папка содержит тесты для проверки работоспособности приложения.

## Структура тестов

### Unit тесты
- `app/src/test/java/com/trusttheroute/app/` - Unit тесты для ViewModels, Repositories, UseCases

### Integration тесты
- `app/src/androidTest/java/com/trusttheroute/app/` - Интеграционные тесты для UI и базы данных

### Ручные тесты
- `test/manual/` - Чек-листы для ручного тестирования

## Запуск тестов

```bash
# Все unit тесты
./gradlew test

# Все instrumented тесты
./gradlew connectedAndroidTest

# Конкретный тест класс
./gradlew test --tests "com.trusttheroute.app.RouteViewModelTest"
```
